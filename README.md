# Lead-Conveyor

Адаптивный лендинг-пейдж для Лидогенерация для бизнеса.

![Скриншот проекта](images/hero.webp)

## Демо

Живое демо: [Ссылка на развернутый сайт](https://lead-conveyor.vercel.app/)

## Технологии

- HTML5
- CSS3 (Flexbox/Grid, кастомные свойства)
- JavaScript (Vanilla JS)
- Swiper.js (для слайдеров)
- WebP для изображений
- Lazy loading и async decoding
